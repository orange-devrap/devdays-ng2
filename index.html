component

  • classe contenant la logique applicative

  • interagit via API : properties / methods

  • lifecycle hooks (init, check, afterX, destroy)


export class MessagesComponent implements OnInit {

  @Input() roomId: string;
  @Output() selectMsg: EventEmitter = new EventEmitter();

  messages: Observable;

  constructor(private chatService: ChatMessagesService) { }

  ngOnInit() {
    this.messages = this.chatService.getMessagesByRoomId(this.roomId);
  }

  selectMessage(msg: ChatMessage) {
    this.selectMsg.emit(msg);
  }
}

Injection de Dépendances

DI principes

  • 1 Injecteur principal + 1 Injecteur par composant

    • Hérite de l'injecteur parent
    • Possibilité de redéfinir le Service à injecter

  • Utilisation d'annotations en ES6 et des types en TypeScript

  • Services disponibles via le constructeur du composant

Injecteur principal

Provider : useValue, useClass, useFactory (official guide)

//  MyAppComponent
@Component({
    selector: 'my-app',
    template: '

                 
 Welcome to our {{breweryName}}

               
'
})
class MyAppComponent {
    constructor(private breweryService:BreweryService) { 
        this.breweryName = this.breweryService.getBreweryName();
    }
}

// injector service to component
bootstrap(MyAppComponent, [BreweryService]);

// equivalent to
bootstrap(MyAppComponent, [new Provider(HeroService,

Custom pipe

// custom pipe
import {Pipe, PipeTransform} from 'angular2/core';
/*
 * Raise the value exponentially
 * Takes an exponent argument that defaults to 1.
 * Usage:
 *   value | exponentialStrength:exponent
 * Example:
 *   {{ 2 |  exponentialStrength:10}}
 *   formats to: 1024
*/
@Pipe({name: 'exponentialStrength'})
export class ExponentialStrengthPipe implements PipeTransform {
  transform(value:number, args:string[]) : any {
    return Math.pow(value, parseInt(args[0] || '1', 10));
  }
}
