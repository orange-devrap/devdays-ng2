pipes

"Transform displayed values within a template"

  • manipulation de donnée

    • {{ price | currency:'EUR':true }} : 51.30 52.30€

  • Pipes disponibles dans le framework :

UpperCasePipe.ts


@Pipe({name: 'uppercase'})
export class UpperCasePipe implements PipeTransform {
  transform(value: string): string {
    if (isBlank(value)) return value;
    if (typeof value !== 'string') {
      throw new InvalidPipeArgumentError(UpperCasePipe, value);
    }
    return value.toUpperCase();
  }
}

Promise

  • gérer les appels asynchrones

  • meilleure gestion des callbacks

  • traitement d'une valeur

  • appelée si résolue

  • ne peut pas être annulée

Observable

  • stream

  • meilleure gestion des callbacks

  • time based

  • 'lazy' jusqu'à l'appel à subscribe

  • unsubscribe, retry, cancel

  • catch & finally 


// As a Promise
getHeroes (): Promise < Hero[] > {
    return this.http.get(this.heroesUrl)
                    .toPromise()
                    .then (this.extractData)
                    .catch(this.handleError);
}
                    

// As an Observable
getHeroes (): Observable < Hero[] > {
    return this.http.get(this.heroesUrl)
                    .map(this.extractData)
                    .catch(this.handleError);
}
